[SMLabs](https://github.com/SushiSalad/SMLabs)
===
An air-strafing and rope-grappling arena shooter designed in Unreal Engine 4

### Features
* Air-strafing movement 
* Rope grappling
* Weapon firing replication
* Health and damage replication
